# FinGrad SG 🇸🇬

**Your First Paycheck, Sorted** — A personal finance app for Singapore fresh graduates.

## Features
- 🧭 4-step onboarding (name, salary, expenses, goals)
- 📊 Live dashboard with budget tracking & KPIs
- 🏛️ CPF Calculator with OA/SA/MA breakdown
- 🎯 Savings goals tracker (BTO, emergency fund, travel, etc.)
- 💡 Financial education (CPF explainer, contribution rates)

## Deploy to GitHub Pages

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) → **New repository**
2. Name it `fingrad-sg` (or anything you like)
3. Set to **Public**
4. Click **Create repository**

### Step 2 — Upload the file
1. In your new repo, click **Add file → Upload files**
2. Drag and drop `index.html`
3. Commit changes

### Step 3 — Enable GitHub Pages
1. Go to repo **Settings → Pages**
2. Under **Source**, select `Deploy from a branch`
3. Branch: `main`, Folder: `/ (root)`
4. Click **Save**

### Step 4 — Access your live app
Your app will be live at:
```
https://<your-github-username>.github.io/fingrad-sg/
```
It may take 1–2 minutes to deploy.

## Local Testing
Just open `index.html` in any browser — no server needed, no dependencies, no build step.
